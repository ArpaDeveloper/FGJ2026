# Duck V: The Phantom Mask

A stealth action game made for Finnish Game Jam 2026 (FGJ2026).

## Description

"Duck V: The Phantom Mask" is the fifth installment in the legendary series created by DD22 studios.

Step into the boots of Agent Duck, an elite operative on a high-stakes mission to save the pond. In "Duck V: The Phantom Mask", stealth is your greatest weapon and your trusted arsenal of masks provided by headquarters to get you out of difficult situations. Even one wrong move in these murky waters could be your last. Tread carefully agent Duck.

## Built With
